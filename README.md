# ChatTalk v4

Paid astrology chat app with timer + auto balance deduction.

## Features
- ⏱ Timer shows chat duration
- 💰 Balance deducts ₹10 per minute
- 🚫 Chat auto-stops when balance = 0

## Run locally
```bash
npm install
npm start
```

Then open: [http://localhost:3000](http://localhost:3000)
